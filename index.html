How to install ZiApi?

Comme ZiApi utilise uniquement des fichiers HPP, vous n'avez pas besoin d'installer quoique ce soit.
Il vous suffit de télécharger les sources sur github: 

git clone https://github.com/KillianG/ZiApi

Un fois que vous avez fait ça, il vous suffit d'include les fichiers .hpp dans vos class et c'est bon.

Avec CMake:

target_include_directory([EXECUTABLE_NAME] PRIVATE ZiApi)

Avec G++:

g++ [sources] -iquote ./ZiApi
