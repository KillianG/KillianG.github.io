ZiApi: How to install

ven. 25 janvier 2019

How to install ZiApi?

As ZiApi is HPP only, there is no need to install anything.
Simply download the sources from github with the following command: 

git clone https://github.com/KillianG/ZiApi

And now, your only job now is to include ZiApi header files.

With CMake:

target_include_directory([EXECUTABLE_NAME] PRIVATE ZiApi)
